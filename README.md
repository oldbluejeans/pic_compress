# pic_compress
compress "jpg", "jpeg", "png" files to a pre given size in batch 
this application runs under windows environment only by default, for Linux desktop, please update the inbound and outbound folder's code yourself if you are familiar with any linux desktops such as Ubuntu.
the script will create c:\users\yourlogon name\desktop\temp\old and new folders automatically for your first run, please copy all the files which you want to convert the size to a smaller level to the old sub folder at first after the srcipt created the two default folders
run the script again after the original files are copied, the converted files will be saved to the "new" sub folder automatically
